## Choose Based On CV Space:

1. **Tight space constraint** → Use **Short Version** (1-2 sentences)
2. **Standard project section** → Use **Medium Version** or **Bullet Points Format**
3. **Portfolio/detailed CV** → Use **Detailed Version**
4. **Technical resume** → Use **Bullet Points** + **Key Technical Highlights**

---

## Pro Tips for CV:

✅ **DO:**
- Quantify achievements (15-50+ items, 60% cost reduction, 100% success rate)
- Mention specific technologies (React 18, GPT-4o-mini, MongoDB)
- Highlight problem-solving (optimized token limits, fixed parsing errors)
- Show business impact (cost reduction, subscription management)

❌ **DON'T:**
- Use vague terms like "worked on" or "helped with"
- List features without context or impact
- Forget to mention your specific role
- Overcomplicate with unnecessary technical jargon
